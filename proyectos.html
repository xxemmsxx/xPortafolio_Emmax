Proyectos que conectan diseño con propósito

Cada proyecto fue una oportunidad para transformar ideas en experiencias visuales memorables. Aquí te presento una selección que muestra mi enfoque creativo, estratégico y adaptable a distintas marcas y necesidades.

PawChic / Branding para una tienda de mascotas

Desarrollo de identidad visual completa para una marca dedicada a accesorios para mascotas. Incluyó logotipo, paleta de colores, estilo gráfico y diseño editorial. El proyecto buscó reflejar una personalidad amigable y moderna, ideal para conectar con dueños de mascotas que valoran estética, ternura y confianza visual.

Pirata Memo / Diseño editorial para restaurante

Creación de un folleto institucional que presenta la propuesta gastronómica del restaurante a través de una narrativa visual atractiva. El diseño equilibra ilustración, texto y estructura clara para comunicar la esencia de la marca. Se trabajó una estética que refleja el carácter creativo y rebelde del restaurante, con una composición visual que invita al lector a explorar los platillos y sumergirse en una experiencia gráfica inmersiva.

Íntima / Diseño web para consultorio de ginecología

Diseño y desarrollo de un sitio web en Wix para el consultorio ginecológico Íntima. El sitio está enfocado en brindar una experiencia amigable y profesional para las pacientes, con secciones interactivas, opciones para agendar citas y un blog con contenido educativo. Se cuidó cada detalle visual para transmitir confianza, accesibilidad y cuidado integral.
